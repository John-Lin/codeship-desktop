# Codeship menubar

A menubar app of codeship show the building status.

# Screenshot

![codeshipbar](./img/codeshipbar.png?raw=true "codeshipbar")

# Builds the mac app

```sh
npm run build
```

# For developing

Install all dependencies packages needed for Codeship
```sh
npm install
```

Run the app
```sh
npm start
```
